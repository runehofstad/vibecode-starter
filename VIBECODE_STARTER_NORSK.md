## 🚀 Kom i Gang på 3 Minutter

### Steg 1: Bruk GitHub Template
1. Gå til [github.com/runehofstad/vibecode-starter](https://github.com/runehofstad/vibecode-starter)
2. Klikk **"Use this template"**
3. Opprett ditt nye repository

### Steg 2: Klon og Setup
```bash
git clone https://github.com/ditt-brukernavn/ditt-prosjekt.git
cd ditt-prosjekt
npm install
```

### Steg 3: Initialiser med Claude Code
```bash
# Åpne Claude Code og kjør en av disse:
"Initialize as React web app with Supabase"
"Initialize as React Native mobile app"  
"Initialize as Next.js full-stack app"
"Initialize as API backend"
"Analyze and configure project"  # Auto-detect
```

### Steg 4: Legg til i .gitignore
```bash
# Hold prosjektet rent
echo -e "\n# Claude helper files (local only)\nclaude-starter/\n.claude/" >> .gitignore
git add .gitignore
git commit -m "Add claude-starter to gitignore"
```

### Steg 5: Start Utvikling
```bash
npm run dev
```

**Ferdig!** Du har nå:
- ✅ Rent prosjekt-repository (kun din kode i git)
- ✅ Alle 29 Sub-Agents tilgjengelig lokalt
- ✅ Full dokumentasjon i `claude-starter/docs/`
- ✅ Claude Code fullstendig konfigurert og klar
